# Sokoban AI Solver

This project implements an **AI solver for the Sokoban puzzle game** using Best-First-Search algorithms.

## Features

- Solve Sokoban levels automatically
- Support multiple search algorithms:
  - Greedy Best-First Search
  - A* Search
- Select different levels (maps)
- Visualize the solution with character movement and box pushing

## Algorithms Used

- Greedy Best-First Search
- A* Search

## Demo Flow

1. Select a Sokoban level
2. Choose a search algorithm
3. Press Enter to start solving
4. Watch the character automatically move and push boxes to the goal
